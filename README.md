## Tech Stack

- **React 19.1.0** — UI composition
- **TypeScript 5.8.3** — static typing
- **Vite 5.0.0** — build tooling and dev server
- **React Router DOM 7.7.0** — client-side routing
- **SCSS** — modular styling and design tokens
- **Framer Motion 12.23.9** — transitions and animations

## Getting Started

### Prerequisites

- Node.js 18+
- Yarn or npm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd task-wargaming

# Install dependencies
yarn install
# or
npm install
```

### Development

```bash
# Start the dev server
yarn dev
# or
npm run dev
```

The app becomes available at `http://localhost:5173`.

### Build & Preview

```bash
# Production build
yarn build
# or
npm run build

# Preview the production bundle
yarn preview
# or
npm run preview
```
